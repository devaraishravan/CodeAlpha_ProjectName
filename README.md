# Data Redundancy Removal System

## Objective

This project identifies redundant and false positive data, validates new entries, prevents duplicate records, and stores only unique data in the database.

## Features

* Duplicate Detection
* False Positive Detection
* Unique Data Validation
* MySQL Database Integration
* Python-Based Implementation

## Technologies Used

* Python
* MySQL
* MySQL Connector
* RapidFuzz

## Workflow

1. Accept user input
2. Check for duplicate email/phone
3. Detect similar names (False Positive)
4. Insert only unique records
5. Maintain database accuracy

## Output Categories

* Redundant Data
* False Positive Data
* Unique Data
